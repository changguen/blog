- [[도커란?|도커]]에선 기본적으로 `docker logs`를 통해 볼 수 있다.
- 이렇게 보는 로그는 `stdout`, `stderr` 로그이다.
  스프링부트에서 기본 설정된 logback은 콘솔에 출력(stdout)하므로 확인 가능하다.
- 따라서 따로 설정하며 파일로 기록하면 보이지 않는다.
  -> 마운트 볼륨으로 밖으로 꺼내서 사용해야 한다.
```bash
# 전체 로그 확인
docker logs <container_name>

# 실시간 로그 추적 (tail -f와 유사)
docker logs -f <container_name>

# 최근 N줄만 보기
docker logs --tail 100 <container_name>

# 타임스탬프 포함
docker logs -t <container_name>

# 특정 시간 이후 로그
docker logs --since 2024-12-19T10:00:00 <container_name>
```
