## [OSM](https://www.openstreetmap.org/)
- 오픈소스 세계지도이다.
- 사용자들이 기여하며 업데이트된다.
- 원하는 지역의 지도를 데이터로 받아볼 수 있다.
- [geofabrik](https://download.geofabrik.de/asia.html)이란 사이트에서는 OSM 데이터를 원하는 '국가' 단위로 받아볼 수 있다.
- 이 데이터에는 건물, 길, 도로 속성등이 구조화된 데이터로 담겨 있다.
## [OSRM](https://project-osrm.org/)
- OSM 데이터를 기반으로 다음 기능들을 HTTP API로 제공한다.
	- 두 지점 최단 경로 계산, 지오메트리 반환
	- 여러 점 간 거리 행렬 계산
	- gpx 좌표를 실제 도로로 스냅핑
	- ...
- Docker나 직접 빌드하는 방식으로 실행이 가능하다.
- 실행시키기 전에 OSM 데이터를 원하는 방식으로 전처리하는 작업이 필요하다.
	1. `extract` : 도로를 전부 노드와 엣지로 쪼개서 그래프를 만든다.
	2. `partition` : 큰 그래프를 작은 그래프들의 집합(셀)으로 나눈다.
	3. `customize` : 각 집합간 최소 거리를 미리 계산해둔다. <- 이 부분이 속도의 핵심!
